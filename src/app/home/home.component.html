Create Your first App

Your very first guide to create an app and start playing with BlueMax, this guide should get you up and running with a web backend ( Restful API ) using ExpressJS

Reading time: ~ 10mins

Level: Introductory

Extensibility

with the powerful extension system, you can customize the behavior of your app, either by using an existing extension from npm or writing it yourself

Modularity & Design

deliver best quality, scalable, maintainable, production grade code, using well known design patterns, powerful module system, and Typescript.

ExpressJS

full support for Expressjs the node.js most popular framework. which allows you to develop your apps with the help of the ExpressJs ecosystem
